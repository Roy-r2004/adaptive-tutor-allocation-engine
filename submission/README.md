# Submission bundle

This is the self-contained deliverables folder for the Apex AI Engineer assessment.

## What's in here

| File | What it is |
|---|---|
| `SUBMISSION.md` | Cover document — architecture, design decisions, how I used AI tools, what to watch in the recording. **Read this first.** |
| `sample_run_results.json` | The exported structured JSON output (5 records) — the primary deliverable from the brief. |
| `sample_run_summary.md` | Human-readable summary of the same 5 records. |
| `screen-recording.mp4` | End-to-end demo video. **(to be added before sending)** |

The full source code lives on GitHub — see the link in the email.

## Final checklist before sending

- [ ] Drop the screen recording in this folder as `screen-recording.mp4`
      (or `.mov` / `.webm` — whatever your screen recorder produces).
- [ ] If the file is over ~25 MB, upload to Google Drive / Loom / WeTransfer
      instead and replace this README's video row with the share link.
- [ ] Push the final commit to GitHub (the JSON output and `SUBMISSION.md`
      should be in the repo too — not just here).
- [ ] ZIP this `submission/` folder and attach it to the email,
      OR upload the ZIP somewhere and link it.
- [ ] Send the email.
